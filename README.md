## Setup  

1. **Clone the repository**
   ```bash
   git clone https://github.com/RJoshi141/portfolio-v2.git
   cd portfolio-v2
````

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Build for production**

   ```bash
   npm run build
   ```

5. **Deploy to GitHub Pages**

   ```bash
   npm run deploy
   ```
